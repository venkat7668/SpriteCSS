
Sprite+CSS Generator

Sprite+CSS Generator; is a cool utility tool to generate Sprite sheets and its CSS code.

For Sprite Sheet; usually require image-editors like Photoshop, for CSS; code-editors like EditPlus, but with our Sprite+CSS Generator, you can create Sprite sheet and CSS code instantly, without any tools and experience.

What all can you do with this tool?
•	You can generate a Sprite sheet that contains all the images in one sheet/file.
•	Also generate a CSS code with Height, Width, and X, Y values of every image in the sheet.
•	Can choose ‘Transparent’ or ‘Color’ background for Sprite sheet.
•	Can customize ‘Canvas size’ by giving preferred sizes for Width and Height.
